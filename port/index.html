Phishing detector

I designed a phishing detector website to help users identify and avoid phishing threats. The website focuses on providing a secure and user-friendly interface that allows users to check URLs for potential phishing activities.

Phishing Detection: Users can enter a URL to instantly scan for phishing threats and protect their online identity.
User-Friendly Interface: Designed with a clean and intuitive layout for easy navigation and use.
Responsive Design: Ensures the website looks great and functions well on both mobile and desktop devices.

Research: Studied various phishing detection tools and gathered user feedback to understand key requirements.
Prototyping: Created high-fidelity prototypes using Figma to visualize the final design.
User Testing: Conducted sessions to gather feedback and refine the design for optimal usability.

This phishing detector website design project showcases my ability to create secure, user-friendly, and responsive web interfaces. It highlights my skills in research, prototyping, and user testing, aimed at delivering a superior user experience.
